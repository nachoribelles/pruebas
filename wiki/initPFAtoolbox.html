initPFAtoolbox

Initiation of the Possibilistic Flux Analysis Toolbox.

Syntax

initPFAtoolbox()


Description

This scripts initializes the Possibilistic Flux Analysis Toolbox. It adds the toolbox folder to the Matlab path. If YALMIP is not already installed, a copy is add to the path. The optimization solver GLPK is selected, if it is currently installed and is detected. Otherwise, YALMIP is initialized with the available solver.

The optimization solver can be changed using the YALMIP function, for example:

yalmip(‘solver’,’glpk’);

Please read the YALMIP help for details about its options.
